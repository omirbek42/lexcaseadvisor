# LexCaseAdvisor

Платформа анализа судебных и арбитражных кейсов для поддержки корпоративных юридических решений.

См. полное описание продукта в [PRD.md](PRD.md).

## Прототип

`index.html` — интерактивный UI-прототип на React (без сборки, через CDN):
поиск похожих кейсов по описанию фактических обстоятельств, фильтры по юрисдикции и типу спора,
карточка кейса, сравнение до 3 кейсов, имитация загрузки нового документа.

Поиск в прототипе работает на лексическом сопоставлении ключевых слов — в продакшене
заменяется на эмбеддинг-поиск согласно разделу 9 PRD.

### Запуск

Просто откройте `index.html` в браузере — сборка не требуется.
